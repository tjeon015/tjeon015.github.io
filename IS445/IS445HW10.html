Data Summary

This data comes John Hopkins University and it is a record of COVID-19 case and death data from counties across the U.S. It contains numerical data on the number of COVID-19 cases and deaths reported by each county and also represents this data in comparison to the total population of each county in the form of rate per 100,000 residents. The data Iâ€™ve decided to represent below is a visualization of the confirmed COVID-19 cases per county in the selected state.

Visualization Description

This graph allows the user to select the state using a drop-down menu to view the case numbers for each county in that state. The y-axis displays the counties of each selected state and the x-axis displays the number of confirmed case per county. I used a bar chart as I thought that would be the best way to compare and contrast between the different counties and I sorted the y-axis according to the x-axis values so that they would be ordered in descending order. I also displayed the county names on the y-axis instead of the x-axis because I thought that would be easier to read.
